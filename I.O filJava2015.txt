				Класс File
----------------------------------------------
File может представлять собой каталог или конкретный файл. У каталога можно узнать список файлов внутри вызвав метод list(), который вернет массив строк с именами.
(с) - в книге приводится множество примеров вывода списков файлов с использованием различных паттернов, можете ознакомиться самостоятельно
Также возможно создавать новые директории и файлы:
createNewFile():----- создает новый файл по пути, который передан в конструктор. В случае удачного создания возвращает true, иначе false
boolean mkdir():----- создает новый каталог и при удачном создании возвращает значение true
boolean delete():---- удаляет каталог или файл по пути, который передан в конструктор. При удачном удалении возвращает true.
boolean exists():---- проверяет, существует ли по указанному в конструкторе пути файл или каталог. И если файл или каталог существует, то возвращает true, иначе возвращает false
----------------------------------------------
				Ввод/вывод
----------------------------------------------
Абстракция потока - произвольный источник или приёмник данных, который способен производить или получать информацию.
Классы для работы с В/В (ввод-вывод) в java разделины на 2 типа - для ввода(InputStream с методами read) и для вывода (OutputStream с методами write)

Reader и Writer
Эти классы работают с символами. Основная причина появления этих классов - работа с 16 битными символами Юникода.
Для работы с "байтовыми" потоками имеются классы inputStreamReader и InputStreamWriter.
